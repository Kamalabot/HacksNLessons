Start by Reading Code

  • * Read a lot of Code that others have written. Head to visualizing data structures and algorithms through animation (http://visualgo.net) or https://the-algorithms.com/* Implement Algorithms by reading the specification, and then check how it has been implemented.
  • * Solve problems using coded, even simple problems can be broken in to steps and coded from there. The book 57 programming challenges can help
  • * Take visualising data using code. This will teach you many concepts about selection, modification of the variables on the fly
  • * Share what you have created with others who are learning like you and discuss. Be open for new ideas and totally unknown ways.
  • * Maintain a list of real world problems that you want to program. Keep researching on how others have solved it.
  • * Learn about the new libraries and how to use their APIs for solving specific problems. Many times the problem you want to code is written by someone else.
  • * Start learning about the system that are working around you. Like Facebook, twitter, youtube, google maps etc. Many open source applications have the code on github. Start reading the code.
